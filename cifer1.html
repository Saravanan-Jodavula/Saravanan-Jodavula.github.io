Time limit: 2 secs.



  • Indian army is going to do a surprise attack on one of its enemies country. The President of India, the Supreme Commander of the Indian Army will be sending an alert message to all its commanding centers. As enemy would be monitoring the message, Indian army is going to encrypt(cipher) the message using basic encryption technique. A decoding key 'K' (number) would be sent secretly. You are assigned to develop a cipher program to encrypt the message. Your cipher must rotate every character in the message by a fixed number making it unreadable by enemies. Given a single line of string 'S' containing alpha, numeric and symbols, followed by a number '0<=N<=1000'. Encrypt and print the resulting string. Note: The cipher only encrypts Alpha and Numeric. (A-Z, a-z, and 0-9) . All Symbols, such as - , ; %, remain unencrypted. Example Input:

    All-convoYs-9-be:Alert1.
    4

    Example Output:

    Epp-gsrzsCw-3-fi:Epivx5.

    #include< stdio.h>
    #include< string.h>
    int main()
    {
    char s[100000],op[100000]; int k,i; //'s' is used to store the message to be encrypted, 'op' is used to store encrypted message output, 'k' is used to store shift value and 'i' is the looping variable.
     gets(s); // Accept the message to be encrypted.
     scanf("%d",&k); // Accept the shift value (the value by which the characters are to be shifted.
     for(i=0;i {
    if(s[i]>=65&&s[i]<=90) //Check if the character is UPPERCASE or not
     if((s[i]+(k%26))>90)//Check if the shifted value's ascii character is greater than 90 (which is invalid, because there is no UPPERCASE alphabet greater than Z)
     op[i]=64+((s[i]+(k%26))-90); // if yes, subtract 90 from the obtained sum (which will give us the normal shift value), and add it to 64 because after Z, the cycle should begin again from A
    .     else
    op[i]=s[i]+(k%26);// if no, proceed normally
     }
    else if(s[i]>=97&&s[i]<=122)//Check if the character is lowercase or not
     {
    if((s[i]+(k%26))>122)/*Check if the shifted value's ascii character is greater than 122 (which is invalid, because there is no lowercase alphabet greater than z)*/
     op[i]=96+((s[i]+(k%26))-122);// if yes, subtract 122 from the obtained sum (which will give us the normal shift value), and add it to 96 because after Z, the cycle should begin again from a.
     else
    op[i]=s[i]+(k%26);// if no, proceed normally
     }
    else if(s[i]>=48&&s[i]<=57)//Check if the character is a numeric digit or not
     {
    if((s[i]+(k%10))>57)/*Check if the shifted value's ascii character is greater than 57 (which is invalid, because there is no decimal digit greater than 9)*/
     op[i]=47+((s[i]+(k%10))-57);/* if yes, subtract 57 from the obtained sum (which will give us the normal shift value), and add it to 47 because after 9, the cycle should begin again from 0.*/
     else
    op[i]=s[i]+(k%10);// if no, proceed normally
     }
    else
    {
    op[i]=s[i]; //ignore (i.e don't shift) any other kind of character (which means special characters)
     }
    }
    puts(op); //print the output (the encrypted message)
     return 0;
    }
