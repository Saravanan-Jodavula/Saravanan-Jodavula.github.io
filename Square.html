Squared and cubic

Time limit: 5 secs.



  • Write a program that reads an integer N (1 < N < 1000). This N is the number of output lines printed by this program.

    Input

    The input file contains an integer N

    Output

    Print the output according to the given example.


    • InputOutput

    5

    1 1 1
    2 4 8
    3 9 27
    4 16 64
    5 25 125

    Here Goes The Code

    #include< stdio.h >

    int main()

    {

    int n,i; /*'N' is the number up to which the squares and cubes are to be printed. 'i' is the looping variable. */

     scanf("%d",&n); //Accept 'n'

    for(i=1;i<=n;i++)

    {
    printf("%d %d %d\n",i,i*i,i*i*i); //Loop and print the squares and cubes of every number up to N separated by a new line.

    }

    return 0;

    }
