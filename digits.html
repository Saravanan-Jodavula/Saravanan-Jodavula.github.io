Digits

Time limit: 5 secs.



  • Given two integers, n and m, how many digits have nm? Examples:
    2 and 10 - 210 = 1024 - 4 digits
    3 and 9 - 39 = 19683 - 5 digits

    Input

    The input is composed of several test cases. The first line has an integer C, representing the number of test cases. The following C lines contain two integers N and M (1 <= N, M <= 100).

    Output

    For each input test case of your program, you must print an integer containing the number of digits of the result of the calculated power in the respective test case.

    Here Goes The Code

    #include< stdio.h >

    int main()

    {

    long long int n,m,temp,d=0;

    /*long long int is used because of the input constraints. 'n','m','temp', and 'd' are used to store the values of base, index, temporary values, and number of digits respectively. */

    scanf("%lld %lld",&n,&m);// Accept the base and the index

    temp=pow(n,m); // Calculate n^m

    while(temp!=0) //as long as the value of temp does not become equal to zero, keep running the loop.

    {

    temp/=10; // keep dividing n^m by 10

    d++; //And increment the counter variable 'd' to count the digits.

    }

    printf("%lld",d); //Print the number of digits.

    return 0;

    }
