Multiple Adder

Time limit: 5 secs.



  • List all the natural numbers below N that are multiples of x or y. Print the sum of these multiples . .

    Example Input:

    3 5 100

    Example Output:

    2318

    Explanation:

    The multiples of 3 and 5 below 100 are 3,6,9,5,10,12,15 etc.

    The sum of these multiples is 2318

    • Here Goes The Code

    #include< stdio.h >

    int main()

    {

    long int i,sum=0; int x,y,n; /*'x','y','n','sum' are used to store the values of x, y, n and the sum of all multiples of x and y below n respectively. 'i' is used as a looping variable. */

     scanf("%d %d %d",&x,&y,&n); /*Accept X, Y and N. N is the upper bound. Multiples of X and Y should be checked till N.*/

    for(i=1;i<=n;i++)

    {

    if(i%x==0||i%y==0) ////Check if the number is multiple or not.

    sum+=i; //Add it the 'sum' variable.

    }

    printf("%ld",sum);//Print the sum.

    return 0;

    }
